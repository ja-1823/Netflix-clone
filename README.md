# # Netflix Clone 🎬

A simple Netflix landing page clone built using **HTML** and **CSS**. This project replicates the basic design and layout of Netflix's homepage, including a navigation bar, hero section, movie cards, feature section, and footer.

## 📌 Features

- Responsive Netflix-inspired design
- Navigation bar with Netflix logo and Sign In button
- Hero section with background image
- Email subscription input field
- Trending Movies section
- Features section
- Footer section
- Hover effects on movie cards
- Mobile-friendly layout using CSS Media Queries

## 🛠️ Technologies Used

- HTML5
- CSS3

## 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open `index.html`.
4. Run the file using **Live Server** or open it directly in a web browser.
